HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • <h1> should only be used once on each webpage.
  • The <a> anchor element creates links to the same webpage or other webpages.
  • Use semantic elements as much as possible.
  • <section> is a semantic element that defines a section in a document that contains thematic content.

CSS

  • The period preceding a selector's name designates it as a class selector.
  • A margin indicates how much space we want around the outside of an element.
  • Padding indicates how much space we want around the content inside an element.
  • A box shadow is described by X and Y offsets relative to the element, blur and spread radius, and color.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • The <script> element is used before the closing </body> tag to link to an external JavaScript file.
  • Falsy values: 0, -0, On, "", null, undefined, NaN, or a Boolean data type with the value of false.
  • An array is a single variable that stores a group of related data. One array can contain multiple data types.
  • Arrays are 0 indexed.
